📥 Getting Started

How to Install

Four simple steps to get Pyra Client running.

1

Download the installer

Click the download button above and save the PyraClient-Setup.exe file to your computer.

2

Run the setup

Double‑click the installer. Windows SmartScreen may prompt – click "More info" and "Run anyway".

3

Log in or play offline

Pyra supports Microsoft OAuth for online play, or you can jump straight into offline mode – no password required.

4

Pick a version & play

Choose any Minecraft version, install mods with one click, and hit Play. That's it!
